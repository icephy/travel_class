任務卡

➊ 任務說明

  • 發言順序：第２位

  • 情境

    預計入住３天，２間房間（一間單人房、一間三人房）。

  • 你的任務

    與飯店櫃檯人員對話，負責幫大家辦理入住手續。

  • 特別任務

    • ➀、詢問房間是否有 Wifi ，並向櫃檯人員要 Wifi 密碼。
    • ➁、回答 Tina，自己為什麼學日文。

  • 求救機會

    卡關時，舉手說「手伝ってください」來獲得抽取求救卡的機會，僅限使用一次。

➋ 補充單字

  • 練習聽、說單字
# 單字 中譯 聆聽
1 予約（よやく） 預約
2 チェックイン Check in
3 パスポート 護照

➌ 文法句型

  • 練習不同句型
# 句型結構 中譯 例句 播放
1 N + が + あります（か） 有Ｎ（嗎）
  • お金がありますか？
  • 有錢嗎？
2 場所 + に + N + がありますか？ 在場所有Ｎ嗎
  • ホテルに食べ物がありますか？
  • 飯店裡有吃的嗎？
3 N + から + N + まで 從～～到～～
  • ９時から１２時まで仕事をします。
  • 我從早上 9 點工作到中午 12 點。

➍ 可能形

  • 運用可能形
Ⅰ 話
さ ない 否定形
し ます ます形
す - 辞書形
せ る 可能形
そ - -
Ⅱ 話
べ ない 否定形
べ ます ます形
べ る 辞書形
べ られる 可能形
べ - -
