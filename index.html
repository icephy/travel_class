又要不能改了嗎

默默反覆複習可能需要用到的日文後，旅客試圖讓自己在與櫃檯人員交流時更加自信...

  • 情境

    預計入住３天，２間房間（一間單人房、一間三人房）。

  • 發言順序

    第２位。

  • 你的任務

    與飯店櫃檯人員對話，負責幫大家辦理入住手續。

  • 特別任務

    • ➀、詢問房間是否有 Wi-Fi，並向櫃檯人員要 Wi-Fi 密碼。
    • ➁、回答 Tina，自己為什麼學日文。

  • 求救機會

    卡關時，舉手說「手伝ってください」來獲得抽取求救卡的機會，僅限使用一次。

同學可以先閱讀「任務卡」，根據文法句型、單字來組成句子，試著完成對話吧！

➋ 文法句型

  • 練習不同句型
# 句型結構 中譯 例句 播放
1 N + が + あります（か） 有Ｎ（嗎）
  • お金がありますか？
  • 有錢嗎？
2 場所 + に + N + がありますか？ 在場所有Ｎ嗎
  • ホテルに食べ物がありますか？
  • 飯店裡有吃的嗎？
3 N + から + N + まで 從～～到～～
  • ９時から１２時まで仕事をします。
  • 我從早上 9 點工作到中午 12 點。
